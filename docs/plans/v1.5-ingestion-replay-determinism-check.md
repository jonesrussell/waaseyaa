# v1.5 Ingestion Replay Determinism Check (#163)

## Goal
Verify byte-for-byte determinism of `ingest:run` outputs and diagnostics across replay runs before closing `#163`.

## Method
- Generated a temporary cross-fixture corpus with four scenarios:
  - `structured_valid`
  - `structured_invalid`
  - `unstructured_valid`
  - `unstructured_invalid`
- For each scenario, executed `ingest:run` twice with identical inputs and explicit deterministic options:
  - fixed `--batch-id`
  - fixed `--timestamp`
  - fixed `--policy`
  - fixed `--source`
- Compared SHA-256 hashes of:
  - mapped output file (`--output`)
  - diagnostics file (`--diagnostics-output`)
