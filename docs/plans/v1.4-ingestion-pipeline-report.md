# v1.4 Ingestion Pipeline Report (#160)

## Scope delivered
- Added deterministic ingestion command: `ingest:run`.
- Supports both input classes:
  - structured (`--format structured`) JSON with `items[]`
  - unstructured (`--format unstructured`) text/notes blocks
- Added deterministic mapping into content/workflow/relationship contracts:
  - normalized stable node keys
  - workflow-state to publication status mapping (`published => 1`, others `0`)
  - deterministic relationship keys (`from_to_type`) and sorted output
  - source audit metadata (`source_ref`) on mapped rows
- Added explicit validation and failure diagnostics:
  - parse/mapping errors under `diagnostics.errors`
  - partial-row warnings under `diagnostics.warnings`
  - non-zero exit when hard mapping errors occur
- Added artifact output controls:
  - `--output` for mapped payload
  - `--diagnostics-output` for standalone diagnostics audit file
- Registered command in `ConsoleKernel` and documented ingestion playbook in operations runbook.
