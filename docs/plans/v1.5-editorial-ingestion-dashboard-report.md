# v1.5 Editorial Ingestion Dashboard Report (#167)

## Goal
Provide deterministic editorial surfaces for ingestion queue status, validation outcomes, inference review load, lifecycle/refresh state, and workflow correctness.

## Delivered
- Added new CLI surface: `ingest:dashboard`.
- Implemented deterministic aggregation over ingest artifacts:
  - queue status (`blocked|review|ready`)
  - schema/validation/runtime failure visibility
  - inference review pending totals
  - refresh category visibility
  - workflow correctness mismatches
- Added JSON contract and human-readable browseable output.
- Wired command into `ConsoleKernel`.
- Added contract spec:
  - `docs/specs/ingestion-editorial-dashboard-contract.md`
- Updated operations playbook with dashboard flows:
  - `docs/specs/operations-playbooks.md`
